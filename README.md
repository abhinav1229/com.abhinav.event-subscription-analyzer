# Event Subscription Analyzer

An automated editor utility for Unity to find missing event unsubscriptions, double subscriptions, and potential memory leaks across your code base.

## Installation via Unity Package Manager (UPM)

1. Open your Unity Project.
2. Navigate to **Window** > **Package Manager**.
3. Click the **+** (plus) button in the top-left corner and select **Add package from git URL...**
4. Paste the following URL:
   `https://github.com/abhinav1229/com.abhinav.event-subscription-analyzer.git`
5. Click **Add**.

*To lock onto a specific version, append the version tag to the URL like this:*
`https://github.com/abhinav1229/com.abhinav.event-subscription-analyzer.git#1.0.0`
